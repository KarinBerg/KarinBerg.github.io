Binding library for HEREMaps

Short description:

Xamarin Binding library to make HEREMaps Premium Edition SDKs (v.3.15) available for Xamarin.iOS and Xamarin.Android Apps. Latest project which tries to use the native iOS and Android HEREMaps SDK inside of an Xamarin app. First version looks promesing but I’m still in testing phase :)

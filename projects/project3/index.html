Microsoft AppCenter Exporter

Short description:

Prometheus exporter for Microsoft AppCenter information e.g. the ui test runs. This project is a copy of the Azure DevOps Exporter which I adjusted to scrape information from the Microsoft AppCenter instead of MicrosoftAzure DevOps. We use it to alert us in case of failing ui tests of our apps.
